# KDE Linux
This is the repository for KDE Linux, codenamed "Project Banana": a KDE-owned general-purpose Linux®-based operating system.

The goal of KDE Linux is to create a bulletproof OS suitable for normal people, showcasing the best of KDE that we can proudly recommend to users and OEMs, with a coherent "here's how you get it" story.


## Current state
KDE Linux has released the *Testing Edition*, which provides built-from-git-master versions of Plasma and KDE apps. **General daily driving should be usable for adventurous people with backups, but not yet for regular people.** See https://linux.kde.org for more details.

Note that the [QA & testing infrastructure](https://invent.kde.org/kde-linux/os-autoinst-distri-kdelinux/-/work_items/2) needs more thoroughness and automation, so expect some bad builds from time to time that you'll need to roll back.

Finally, anything currently pre-installed may be removed before the beta release. Don't build mission-critical workflows around any of it.

## Use it
Life on a cutting-edge immutable OS can be unfamiliar! Read the [official documentation](https://linux.kde.org/docs) to learn how to perform various tasks.


## Get help with it
Post at https://discuss.kde.org and tag your post with "kde-linux".


## Report an issue with it
- Issues in the OS itself: https://invent.kde.org/kde-linux/kde-linux/-/issues/new (ignore the scary red banner at the top of the page)
- Issues in any of the KDE software included with the OS: https://bugs.kde.org/


## Contribute to it
The KDE Linux team is super excited to receive new contributors! The OS is easy to hack on, and the developers hang out in the [#kde-linux:kde.org Matrix room](https://matrix.to/#/%23kde-linux:kde.org).

See https://linux.kde.org/docs/kde-linux-dev for details.
